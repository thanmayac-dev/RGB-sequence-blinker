# RGB Sequence Blinker

A beginner Arduino project that blinks a red LED 5 times, 
green 10 times, and blue 15 times in sequence using for loops.

## Components
- Arduino Uno 
- Red, Green, Blue LEDs
- 3x 3300 ohm resistors
- Breadboard

## Pins Used
- Pin 13 → Red LED
- Pin 12 → Green LED
- Pin 8  → Blue LED
